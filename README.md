# TAEL Demo

Static clickable demo for the TAEL private project financing protocol.

## Run locally

\`\`\`bash
cd projects/tael-demo
python3 -m http.server 4173
\`\`\`

Open \`http://localhost:4173\`.

## Deploy to Vercel

\`\`\`bash
cd projects/tael-demo
vercel --prod
\`\`\`

This repo currently has no authenticated Vercel session on this machine, so deployment requires \`vercel login\` or a token.
